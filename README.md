# WorldCup 2026 Prediction

سامانه پیش‌بینی جام جهانی ۲۰۲۶ با رابط فارسی، پنل ادمین، پیش‌بینی چندشرطی، امتیازدهی خودکار، مدیریت جوایز، بنرهای تبلیغاتی، بکاپ و بازیابی، و سینک نتایج زنده و گذشته.

## Features

- رابط کاربری ریسپانسیو برای صفحه اصلی، صفحه بازی، حساب کاربری و پنل ادمین
- ورود یکپارچه کاربر و ادمین با شماره موبایل
- پنل ادمین برای مدیریت:
  - بازی‌ها
  - شرط‌های هر بازی
  - شرط‌های پیش‌فرض
  - نتایج مسابقه
  - کاربران و خروجی اکسل
  - جوایز و کدهای تخفیف
  - بنرهای سایت و تبلیغات
  - تنظیمات برندینگ
  - بکاپ و بازیابی
- امتیازدهی خودکار به پیش‌بینی‌ها بعد از ثبت نتیجه
- محدودیت زمانی پیش‌بینی:
  - تا ۱۰ دقیقه قبل از شروع بازی
  - فقط تا ۴۸ ساعت مانده به بازی
- نمایش نتایج زنده و وضعیت مسابقه
- جوایز قابل خرید با امتیاز و نمایش کد تخفیف برای کاربر
- خروجی کاربران به صورت CSV سازگار با Excel
- آپلود تصویر برای بنرها و جوایز

## Tech Stack

- PHP
- MySQL / MariaDB
- HTML / CSS / JavaScript
- بدون وابستگی فرانت‌اند سنگین

## Project Structure

```text
.
├── admin/            # پنل ادمین
├── assets/           # استایل، اسکریپت مشترک، لوگو
├── backend/          # API، منطق کسب‌وکار، بکاپ، نصب و آپلود
├── install/          # نصب اولیه و ساخت config.php
├── knockout/         # صفحه نمودار حذفی
├── match/            # صفحه هر بازی
├── profile/          # حساب کاربری
└── index.html        # صفحه اصلی
```

## Installation

### 1. Clone

```bash
git clone https://github.com/YOUR_USERNAME/worldcup-2026-prediction.git
cd worldcup-2026-prediction
```

### 2. Prepare Web Server

پروژه را داخل یک وب‌سرور PHP مثل Apache, Nginx + PHP-FPM, MAMP, XAMPP یا Laragon قرار دهید.

نقطه شروع پروژه:

- `/index.html`
- نصب اولیه: `/install/`

### 3. Create Database

یک دیتابیس MySQL یا MariaDB بسازید.

### 4. Run Installer

مرورگر را روی این آدرس باز کنید:

```text
/install/
```

در صفحه نصب:

1. اطلاعات دیتابیس را وارد کنید
2. `config.php` ساخته می‌شود
3. جداول ساخته می‌شوند
4. ادمین اولیه ثبت می‌شود
5. برنامه اولیه مسابقات و شرط‌های پیش‌فرض بارگذاری می‌شوند

## Configuration

فایل `config.php` در زمان نصب ساخته می‌شود و داخل ریپو نگه‌داری نشده است.

نمونه تنظیمات:

```php
<?php
return [
    'db_host' => 'localhost',
    'db_name' => 'worldcup',
    'db_user' => 'root',
    'db_pass' => '',
    'db_charset' => 'utf8mb4',
    'table_prefix' => 'wc_',
];
```

## Main Admin Capabilities

### Matches

- افزودن، ویرایش و حذف بازی
- تعیین مرحله، گروه، زمان، مکان، وضعیت
- ثبت نتیجه دستی
- اتصال به شناسه منبع خارجی برای سینک خودکار

### Bets

- تعریف شرط اختصاصی برای هر بازی
- تعریف شرط‌های پیش‌فرض برای همه بازی‌ها
- همگام‌سازی شرط‌های پیش‌فرض با بازی‌ها

### Branding

- لوگوی اصلی، نوبار، مودال ورود، فوتر و پنل ادمین
- بنر اصلی هیرو
- بنر جایگزین بخش کناری صفحه اصلی
- بنرهای تبلیغاتی زیر جدول امتیازات

### Rewards

- تعریف جایزه با:
  - عنوان
  - توضیح
  - تصویر
  - کد تخفیف
  - لینک خرید محصول
  - امتیاز لازم
  - موجودی
- هر کاربر فقط یک‌بار می‌تواند هر جایزه را دریافت کند

### Data Operations

- بکاپ JSON / ZIP
- بازیابی بکاپ
- خروجی CSV کاربران

## Live Scores

پروژه برای سینک خودکار نتایج و وضعیت زنده از چند provider پشتیبانی می‌کند:

- Varzesh3 HTML
- ESPN Scoreboard JSON
- Generic JSON Feed

تنظیم provider و URL از داخل پنل ادمین انجام می‌شود.

## Security Notes

- `config.php` را داخل Git commit نکنید
- اگر از آپلود عمومی استفاده می‌کنید، دسترسی پوشه `uploads/` را مدیریت کنید
- برای محیط production، لاگ خطا و دسترسی سرور را محدود کنید

## Recommended .gitignore

این پروژه فایل‌های زیر را داخل GitHub نگه نمی‌دارد:

- `config.php`
- `uploads/`

- لاگ‌ها و فایل‌های موقت
