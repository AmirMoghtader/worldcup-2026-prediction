بکاپ و بازگردانی

خروجی بکاپ

یک فایل بکاپ از دیتابیس همین پروژه بگیر. فرمت ZIP در صورت پشتیبانی سرور ساخته می‌شود و در غیر این صورت خروجی JSON دانلود می‌شود.

بازگردانی بکاپ

فایل بکاپ همین پروژه را انتخاب کن و بازگردانی را انجام بده. این عملیات داده‌های فعلی را جایگزین می‌کند.

تنظیمات برندینگ و پیش‌بینی

راهنمای عکس‌ها: لوگوی اصلی بهتر است `512x512` باشد. آیکن مرورگر و جستجوی گوگل بهتر است `512x512` یا `192x192` و مربع باشد. لوگوی نوبار، ورود، فوتر و پنل بهتر است `256x256` و PNG شفاف باشند. بنر هیرو: برای دسکتاپ سایز پیشنهادی `1600x360` و برای موبایل `900x320` است. اگر خالی بماند، هیروی شیشه‌ای پیش‌فرض سایت نمایش داده می‌شود.
برندینگ و سئو: اگر در فیلد نام برند مثلا `آنوبز` وارد شود، عنوان صفحه‌ها، توضیحات متا، favicon، و لینک‌های sitemap با همان هویت برند برای کل سایت ساخته و به‌روز می‌شوند.
بنر هیرو صفحه جوایز: برای دسکتاپ سایز پیشنهادی `1600x360` و برای موبایل `900x320` است. به محض آپلود این بنر، هیروی صفحه جوایز فقط خود بنر را نشان می‌دهد و متن و اطلاعات دیفالت دیگر نمایش داده نمی‌شود.
اگر این فیلد خالی بماند، با کلیک روی بنر خود تصویر باز می‌شود.
اسلایدر جوایز بالای نفرات برتر: به صورت خودکار از اولین جوایز فعال نمایش داده می‌شود. اینجا فقط تعداد اسلایدهای قابل نمایش را مشخص می‌کنید. بنر ویژه پایین نفرات برتر هم با سایز پیشنهادی `1200x1200` از همین بخش تنظیم می‌شود.
پاپ‌آپ ورودی سایت: بعد از ورود کاربر به صفحه اصلی نمایش داده می‌شود. سایز پیشنهادی عکس `1200x1500` یا نزدیک به آن است تا داخل مودال تمیز دیده شود.
برای سینک خودکار نتیجه و لایو، می‌توانی از `Varzesh3` یا یک API JSON استفاده کنی. برای `Varzesh3` همین لینک جواب می‌دهد: `https://www.varzesh3.com/livescore` و هر ۵ دقیقه خوانده می‌شود. برای JSON هم اگر برای هر بازی `شناسه منبع` ثبت شود، سینک دقیق‌تر می‌شود.
