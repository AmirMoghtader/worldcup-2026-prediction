بکاپ و بازگردانی

خروجی بکاپ

یک فایل بکاپ از دیتابیس همین پروژه بگیر. فرمت ZIP در صورت پشتیبانی سرور ساخته می‌شود و در غیر این صورت خروجی JSON دانلود می‌شود.

بازگردانی بکاپ

فایل بکاپ همین پروژه را انتخاب کن و بازگردانی را انجام بده. این عملیات داده‌های فعلی را جایگزین می‌کند.

تنظیمات برندینگ و پیش‌بینی

هویت برند و سئو

نام برند، نام سایت و متن‌های اصلی از اینجا کنترل می‌شوند و روی عنوان صفحه‌ها، توضیحات متا و برندینگ کلی پروژه اثر می‌گذارند.

لوگوها و آیکن‌ها

لوگوی اصلی بهتر است `512x512` باشد. آیکن مرورگر و جستجوی گوگل بهتر است مربع و ترجیحا `512x512` یا `192x192` باشد. لوگوهای نوبار، ورود، فوتر و پنل بهتر است `256x256` و PNG شفاف باشند.

نوار باریک بالای سایت

این نوار به صورت پیش‌فرض نمایش داده نمی‌شود. اگر عکس بگذارید، به صورت یک بنر باریک و کلیکی بالای سایت ظاهر می‌شود. سایز پیشنهادی دسکتاپ `1600x80` و موبایل `960x96` است.

بنر هیرو صفحه اصلی

سایز پیشنهادی دسکتاپ `1600x360` و موبایل `900x320` است. اگر خالی بماند، هیروی شیشه‌ای پیش‌فرض سایت نمایش داده می‌شود.

بنر هیرو صفحه جوایز

سایز پیشنهادی دسکتاپ `1600x360` و موبایل `900x320` است. با فعال شدن این بنر، متن و اطلاعات دیفالت هیروی صفحه جوایز حذف می‌شود.

اسلایدر جوایز و بنر ویژه صفحه اصلی

اسلایدر بالای نفرات برتر از اولین جوایز فعال خوانده می‌شود. بنر ویژه پایین آن هم از همین باکس مدیریت می‌شود. سایز پیشنهادی بنر ویژه `1200x1200` است.

پاپ‌آپ شروع سایت

بعد از ورود کاربر به صفحه اصلی نمایش داده می‌شود. سایز پیشنهادی تصویر `1200x1500` یا نزدیک به آن است تا داخل مودال تمیز دیده شود.

سینک نتایج زنده و ثبت نتیجه

برای سینک خودکار نتیجه و لایو می‌توانی از `Varzesh3` یا یک API JSON استفاده کنی. برای `Varzesh3` همین لینک جواب می‌دهد: `https://www.varzesh3.com/livescore` و هر ۵ دقیقه خوانده می‌شود.
